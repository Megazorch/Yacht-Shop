About Project

Yacht-Shop (or Y-Shop) is created by Roman Ivakhiv.

    Technologies used:
  • Python 3.11
  • Django 4.2
  • Django REST framework
  • PostgreSQL 15

HTML template is 100% free and provided by TemplateMo website.

Some CSS code was writen locally to adopt for a new features.

A new Javascript function was writen, which sends AJAX request to the endpoint.

Web site is running on Digital Ocean IaaS using Ubuntu 20.04, Gunicorn and Nginx.
